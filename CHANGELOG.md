# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-27

### Added
- Initial release of Student Card Generator
- Support for 12 countries (India, US, UK, Canada, Australia, Germany, France, Korea, Japan, Singapore, China, Brazil)
- Database of 60 real universities with authentic domains
- Random student data generation (names, DOB, student ID, department, email)
- Professional ID card design matching real student cards
- PNG export functionality (high quality, 3x scale)
- PDF export functionality (standard CR80 size)
- Email field with copy-to-clipboard feature
- Avatar selection from placeholder images
- Photo upload with drag-and-drop support
- Real-time card preview
- Responsive design for mobile and desktop
- Multi-language support (English, Hindi, Korean, Japanese, Chinese)
- Font support for Asian languages (Noto Sans JP/KR)
- Buy Me a Coffee donation link
- Clean glassmorphism UI design
- Smooth animations and transitions

### Technical
- Single-file HTML application for easy deployment
- Tailwind CSS for styling (CDN)
- Vanilla JavaScript (no frameworks)
- html2canvas for PNG export
- jsPDF for PDF export
- Google Fonts integration

### Documentation
- Comprehensive README.md
- MIT License
- Contributing guidelines
- .gitignore file
- Complete project walkthrough
